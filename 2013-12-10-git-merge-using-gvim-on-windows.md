## 工具配置 

假定 git 安装在 `D:\Git` 文件夹下

1. 复制 D:\Git\bin\vim 为 gvim，内容如下

    #!/bin/sh

    exec /D/Vim/vim74/gvim "$@"

2. 修改 D:/Git/etc/gitconfig 加入如下配置

    [merge]
        tool = gvimdiff

## 合并操作

运行 `git mergetool`
